# E-DUC: Educational Data Search for Cities in Bahia.

## Objective: Provide a centralized platform to query elementary school educational data for cities of Bahia, Brazil.

## Featured Cities: Alagoinhas, Feira de Santana, Salvador, Catu, Aramari, Pojuca, Entre Rios, Inhambupe, Araçás. 

# Technologies Used:
## 🖥️ Backend: 
-  Node.js
-  Express
## 🌐 Frontend
- JavaScript
- CSS
- HTML

## 🔌 Integrations
- Axios
- QEdu API

 ## 🔐 Middleware
 - CORS

 ## ☁️ Deployment
 - Render: [Access the project](https://e-duc.onrender.com/)

## 🚀 Features
- 📈 Educational data visualization  
- 🔍 Search by city and year  
- 🌐 Integration with external API
